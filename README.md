# DrillChain
A keychain shaped to look like a big drill that uses a bistable multivibrator circuit to toggle the mini motor disc off and on.

- 1 Battery holder
- 2 NPN tranistors
- 2 20Ω resistors
- 2 700Ω resistors
- 2 1kΩ resistors
- 1 mini motor disc
- 2 push buttons
